# Weekly Tasks - JustLab AI

Team task assignments and progress tracking.

- [View Team Contributions](https://github.com/justlab-ai/weekly-tasks/graphs/contributors)
- [Current Week Tasks](tasks/)
- [Submit Updates](updates/)

## How It Works

### For Task Assigners
1. Create a new folder: `tasks/YYYY-WXX/` (e.g., `tasks/2025-W49/`)
2. Copy `templates/task-template.md` to `tasks/YYYY-WXX/tasks.md`
3. Fill in tasks and assignees

### For Team Members
1. Clone the repo (first time only):
   ```bash
   git clone https://github.com/justlab-ai/weekly-tasks.git
   ```
2. Pull latest changes:
   ```bash
   git pull
   ```
3. Create your update file: `updates/YYYY-WXX/[your-name].md`
4. Use the template from `templates/update-template.md`
5. Commit and push:
   ```bash
   git add .
   git commit -m "Update: [your-name] Week XX"
   git push
   ```

## Structure

```
weekly-tasks/
├── tasks/           # Weekly task assignments
│   └── YYYY-WXX/
│       └── tasks.md
├── updates/         # Team member progress
│   └── YYYY-WXX/
│       └── [name].md
└── templates/       # Templates
```
